# GestureForge 3D Sound Sources

The app currently synthesizes interaction sounds with Web Audio so the prototype is self-contained and legal to ship.

Production replacement sources should use royalty-free clips from:
- Pixabay Sound Effects: https://pixabay.com/sound-effects/
- Freesound Creative Commons 0 filter: https://freesound.org/search/?f=license:%22Creative+Commons+0%22

Drop normalized `.mp3` files in this folder using the names in the master prompt. `SoundEngine.js` will continue to work procedurally if an asset is unavailable.
